# 文字轉語音
透過 Web API 中的 `SpeechSynthesis` 介面，將輸入的文字轉為聲音。

## 可用功能
* 輸入的文字轉為聲音。
* 調整發音的語言（支持語言：中、日、英）。

## 預計功能
* 調整發音的音量。
* 調整發音的音調。
* 調整發音的速度。

## 已知問題
* 僅支持 PC 版 Microsoft Edge 瀏覽器，其於瀏覽器出現無法載入語音包的問題。
