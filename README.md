# 文字轉語音
透過 Web API 中的 `SpeechSynthesis` 介面，將輸入的文字轉為聲音。

## 功能
* 調整發音語系及聲音。
* 調整發音音量。
* 調整發音語速。
* 輸入的文字轉為聲音。
